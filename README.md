# Minimal Python Project

This example is set up so it can run both locally and in a GitHub Codespace.

## Run locally

```bash
python app.py
```

## Run in Codespaces

1. Open the repository in GitHub Codespaces.
2. Rebuild the container if prompted.
3. Run:

```bash
python app.py
```

> The dev container now includes Git LFS, so pushes that rely on Git LFS should work without needing a separate local install.
